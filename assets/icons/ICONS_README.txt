PWA Icons Required
==================

Please add the following icon files to this directory:

1. icon-192x192.png
   - Size: 192x192 pixels
   - Format: PNG with transparency
   - Purpose: Android home screen icon

2. icon-512x512.png
   - Size: 512x512 pixels
   - Format: PNG with transparency
   - Purpose: Android splash screen and high-res displays

Design Guidelines:
- Use housekeeping/cleaning theme (e.g., bed, spray bottle, checklist)
- Primary color: #2196F3 (blue)
- Include hotel/hospitality elements
- Ensure icon works well at small sizes
- Add safe zone padding (10% margin from edges)

You can create these using:
- Canva (https://www.canva.com)
- Figma (https://www.figma.com)
- Adobe Illustrator/Photoshop
- Or use icon generators like https://realfavicongenerator.net/
